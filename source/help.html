Cadasto de produtos
  • Como usar

    Iniciando o servidor

  1. Instale o JSON Server com o comando " npm install -g json-server " no prompt de comando.

  2. Abra o prompt de comando no caminho do arquivo "DB.JSON"


  3. Execute o comando " json-server --watch db.json " para startar o servidor.


    Entendendo API

    A API é dividida em dois campos, o campo para cadsatrar os produtos e outro para ver produtos já cadastrados:

    É possivel realizar as requisições POST "Enviar", PUT "Editar" e DELETE "Apagar".
